O que é HTML?

(voltar)

HTML, apesar de receber ser usualmente chamada de linguagem, não é uma linguagem de programação como pode fazer muitos iniciantes acharem que é. Na verdade, trata-se de uma linguagem de marcação, muito usada para a construção de páginas web.

O HTML compõe o esqueleto de uma página web. É ele que define a estrutura em sua forma crua, como podemos ver nesta página que está sendo construída agora. Para ir além do que já podemos ver com esta linguagem de marcação, são usados o JavaScript, esta sim é uma linguagem de programação, compondo a interação de uma página o usuário, para além dos elementos estáticos. E também o CSS, sendo considerada uma linguagem de design gráfico. Este por sua vez responsável pela estilização da página web.

Tags

As tags thml fazem parte, obviamente, da linguagem de marcação HTML. A função das tags é informar ao navegador a estrutura ques está sendo inserida e montada. Podendo ser imagens, títulos, citações, vídeos, pequenas animações (gifs, por exemplo) e entre outros. Você pode conferir algumas tags clicando aqui!
